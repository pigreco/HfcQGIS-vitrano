# `array_avg`

Calcola il valore medio di un array.

## Sintassi

array_avg(array)

## Argomenti

* _array_ un array o stringa (valori separati da virgola) 

## Esempi

* `array_avg(array(1,2,3)) → 2`
* `array_avg('1,2,3') → 2` dalla versione 1.4 del plugin!!!
