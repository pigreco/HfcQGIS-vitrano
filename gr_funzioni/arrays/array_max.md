# `array_max`

Restituisce il valore massimo di un array.
## Sintassi

array_max(_array_) 

## Argomenti

* _array_ un array o stringa (valori separati da virgola) 

## Esempi

* `array_max(string_to_array( 'a,b,c,a,a')) → 'c'`
* `array_max( array( 11,2,45,2345,123)) → 23459`
* `array_max('a,b,c,a,a') → 'c'`
