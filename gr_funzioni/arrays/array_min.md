# `array_min`

Restituisce il valore minimo di un array.

## Sintassi

array_min(_array_) 

## Argomenti

* _array_ un array o stringa (valori separati da virgola)

## Esempi

* `array_min(string_to_array( 'a,b,c,a,a')) → 'a'`
* `array_min( array( 11,2,45,2345,123)) → 2`
* `array_min('a,b,c,a,a') → 'a'`
